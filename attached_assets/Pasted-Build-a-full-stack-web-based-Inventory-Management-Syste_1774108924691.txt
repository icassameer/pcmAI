Build a full-stack web-based Inventory Management System with the following complete specifications:

---

## 🏗️ TECH STACK
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express.js
- **Database**: PostgreSQL with Drizzle ORM
- **Auth**: JWT + bcrypt (role-based access control)
- **PDF Generation**: pdfkit or puppeteer (for GST invoices & reports)
- **Excel Export**: exceljs (for reports)

---

## 👥 USER ROLES & PERMISSIONS

### Roles:
1. **Super Admin** – Full access: manage users, settings, billing, all reports
2. **Admin** – Manage products, purchases, sales, GST reports, view all data
3. **Store Keeper** – Add/update stock, record purchases, view own entries
4. **Accountant** – View/export reports, billing, GST summaries only
5. **Viewer** – Read-only access to stock and reports

### Security Requirements:
- JWT authentication with refresh tokens (access token: 15min, refresh: 7 days)
- Passwords hashed with bcrypt (salt rounds: 12)
- Role-based route guards on both frontend and backend
- Rate limiting on all API endpoints (express-rate-limit)
- SQL injection prevention via parameterized queries (Drizzle ORM)
- XSS protection via helmet.js
- CORS configured for allowed origins only
- Input validation on all forms (zod schema validation)
- Audit log table: track every create/update/delete with user ID + timestamp
- Session invalidation on logout (token blacklist via Redis or DB)
- Account lockout after 5 failed login attempts (unlock after 30 min)

## 📦 PRODUCT MANAGEMENT

### Product Fields:
- Product ID (auto-generated, unique)
- Product Name
- Category (custom categories, user-defined)
- Brand / Manufacturer
- HSN Code (for GST)
- Unit of Measurement: **gm, kg, ltr, ml, pcs, tonne, meter, feet, box, dozen, pack**
- Purchase Price (per unit)
- Selling Price (per unit)
- MRP
- GST Rate (0%, 5%, 12%, 18%, 28%)
- IGST / CGST / SGST auto-split
- Current Stock Quantity
- Minimum Stock Alert threshold
- Shelf / Rack / Location in warehouse
- Barcode / SKU
- Product Image upload
- Supplier/Vendor linked
- Expiry Date (optional, for perishables)
- Active / Inactive status

### Features:
- Bulk product import via CSV/Excel upload
- Export product list to Excel/PDF
- Low stock alert dashboard widget
- Search, filter, sort, paginate (handle 100,000+ products efficiently — use indexed DB queries, server-side pagination)
- Product history log (price changes, stock adjustments)

## 🛒 PURCHASE MANAGEMENT

- Record purchases from suppliers
- Fields: Supplier, Invoice No, Invoice Date, products (multiple line items), quantity, unit, purchase price, GST, discount, total
- Auto-update stock on purchase save
- Purchase return / credit note
- Pending payments tracker
- Export purchase register (monthly/annual) to Excel/PDF

## 💰 SALES MANAGEMENT

- Create sale / invoice for customers
- Fields: Customer name, phone, address, GSTIN (optional), sale date, products (multi-line), quantity, unit, rate, discount, GST breakdown, grand total
- Auto-deduct stock on sale
- Sales return / debit note
- Partial payment / balance due tracking
- Export sales register (monthly/annual) to Excel/PDF

---

## 🧾 GST BILLING

- Generate GST-compliant tax invoices (as PDF)
- Invoice contains:
  - Seller details (name, address, GSTIN, state)
  - Buyer details
  - Serial invoice number (auto-increment, format: INV-2024-0001)
  - HSN/SAC code per item
  - Taxable value, CGST, SGST, IGST (auto-split based on inter/intra state)
  - Total in words
  - E-way bill number field (optional)
  - Digital signature placeholder
- GSTR-1 style summary report (B2B, B2C breakdowns)
- Monthly GST summary (Input tax credit vs Output tax)
- Download invoice as PDF
